### Sección de Postulación Directa (Tarjeta de Formulario)

El formulario se encuentra dentro de un contenedor gris claro con bordes redondeados pronunciados, lo que lo separa visualmente del fondo blanco.

- **Título del Formulario**: **"Postulación Directa"**.
- **Instrucciones**: Texto breve que indica el proceso: *"Envíanos tu CV e indícanos el área a la que deseas aplicar..."*.
- **Campos de Entrada (Inputs)**:
    - Organizados en una grilla de dos columnas.
    - **Nombre**: Input de texto simple.
    - **Correo Electrónico**: Input con validación de email.
    - **Área deseada**: Un menú desplegable (Select) con opciones (ej. Ventas, Logística, Administración).
    - **Número de teléfono**: Input numérico para contacto directo.
- **Área de Mensaje**: Un campo de texto grande (**Carta de Presentación**) para que el postulante se describa.

### 3. Módulo de Carga de Archivos (File Upload)

- **Diseño**: Un recuadro con borde punteado que indica una zona de "Arrastrar y soltar" (Drag & Drop). Al cliquearlo, se abre el explorador de archivos del ordenador para adjuntar archivos.
- **Iconografía**: Icono de subida (flecha hacia arriba).
- **Restricciones Técnicas**: Indica claramente los formatos aceptados (**PDF**) y el peso máximo (**5MB**).

### 4. Botón de Acción (CTA)

- **Estilo**: Botón de ancho completo en color amarillo vibrante.
- **Contenido**: Icono de avión de papel y el texto **"Enviar Postulacion"**.
